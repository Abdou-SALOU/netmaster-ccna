# 🛰️ NetMaster — Plateforme de révision CCNA / CCNP (2ISI)

Plateforme web **interactive et 100 % hors-ligne** pour réviser tout le programme
réseaux avant l'examen : VLANs, OSPF, OSPF multizones, BGP, IPv6 + les 6 TP.

## ▶️ Comment l'ouvrir

**Double-clique simplement sur `index.html`** (ou sur `Ouvrir-la-plateforme.bat`).
Elle s'ouvre dans ton navigateur (Chrome / Edge / Firefox). Aucune connexion Internet
n'est nécessaire, aucune installation.

## 📚 Ce que tu y trouves

- **5 modules de cours** avec explications approfondies, schémas et **animations interactives** :
  - VLANs (802.1Q, trunk, VTP, inter-VLAN, STP)
  - Routage État de Lien — avec l'algorithme **Dijkstra animé pas-à-pas**
  - OSPF Multizones (LSA 1→7, zones Stub / NSSA…)
  - BGP 4 (eBGP/iBGP, AS_PATH, attributs)
  - IPv6 (compresseur d'adresses, EUI-64, en-tête, ICMPv6…)
- **6 travaux pratiques** : VLAN, NAT/PAT, OSPF MZ, IPv6, Tunnel 6in4, BGP — avec toutes les commandes Cisco.
- **🎯 Mode Examen** : 86 questions corrigées, chrono, score final.
- **🃏 Flashcards** : 44 cartes pour mémoriser l'essentiel.
- **⌨️ Mémo commandes** : toutes les commandes IOS classées par thème.
- Suivi de **progression**, thème **clair/sombre**.

## 🧩 Animations interactives incluses

Dijkstra/SPF · segmentation VLAN · trame 802.1Q · tagging trunk · zones OSPF & LSA ·
propagation BGP & anti-boucle · compresseur IPv6 · en-tête IPv6 · EUI-64 ·
table NAT/PAT · tunnel IPv6-in-IPv4.
